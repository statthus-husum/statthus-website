# Hugo Theme Changes

## Overview
How to deal with Theme changes

## Quick Reference
Hugo Themes should not be modified unless there is a compelling reason. Hugo provides mechanisms to override theme codes and settings in an individual layout folder.

## Detailed Guidelines
When a request requires changes of the theme, analyse carefully whether this cannot be changed more easily with an project /layout overlay. Ask the requesting user to choose between the alternatives.
